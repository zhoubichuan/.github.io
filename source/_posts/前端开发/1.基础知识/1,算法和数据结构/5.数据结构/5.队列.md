## 队列

#### 概念

队列是一个线性结构，特点是在某一端添加数据，在另一端删除数据，遵循先进先出的原则。

#### 实现

`单链队列`

```js
class Queue{
	constructor(){
		this.queue=[]
	}
	enQueue(item){
		this.queue.push(item)
	}
	deQueue(){
		return this.queue.shift()
	}
	getHeader(){
		return this.queue[0]
	}
	getLength(){
		return this.queue.length
	}
	isEmpty(){
		return this.getLength()===0
	}
}
```

因为单链队列在出队操作的时候需要O(n)的时间复杂度，所以引入了循环队列。循环队列的出队操作平均是O(1)的时间复杂度。
