### 2.vue.config.js的配置

#### baseurl

修改项目的地址：如`http://localhost:8080/vue/`

```
//vue.config.js
module.exports={
...
baseUrl:'vue',
...
}
```

其实改变的是webapck配置文件中output的`publicPath`
