## 删除文件夹操作
文件目录如下
- c
    - admin
        - my

### 1.删除 admin下面的所有文件和文件夹
在c目录下面打开cmd命令窗口
```
rmdir /s/q admin
```
- /s 是代表删除所有子目录跟其中的档案。 
- /q 是不要它在删除档案或目录时，不再问我 Yes or No 的动作。
### 2.删除指定目录下的文件
```
rmdir /s/q c:\admin\my
```
删除c盘下面的admin目录下面的my
