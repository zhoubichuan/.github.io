## 1.v-for

```
<ul>
    <li v-for="(fruit,index) in fruits">{{index}}{{fruit}}</li>
</ul>
```

## 2.如果不想要最外面的标签

只有一个的情况可以在里面加上 key

```
<template v-for="i in 3">
    <div :key="i">{{i}}</div>
</template>
```

有多个可以标识下不同的 key

```
<template v-for="i in 3">
    <div :key="i+'_i'">{{i}}</div>
    <div :key="i">{{i}}</div>
</template>
```
