## vue 中常用的指令

### `v-once`

```
<div v-once>{{msg}}</div>
```

### `v-html`

```
<div v-html="d"></div>
```

### `v-if` 控制的是 dom，支持 template

```
<template v-if="flag">
    <div>123</div>
</template>
<div v-else>321</div>
```

### `v-show` 控制的是样式，不支持 template

```
<div v-show="flag">hello</div>
```
