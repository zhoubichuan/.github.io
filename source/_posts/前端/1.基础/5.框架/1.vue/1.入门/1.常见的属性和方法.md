## 安装

```
yarn add vue
```

## vue 中常见的属性和方法

* vm.$set()给 data 设置新的值

```
vm.$set(vm.info,'address','hello1')
```

* vm.$watch()监听变化，多次更新只会触发一次

```
vm.$watch('info.xxx',function(newValue,oldValue){
    console.log(newValue,oldValue)
})
vm.info.xxx="nnnnn"
```

* vm.mount()

```
let vm=new Vue({
    data:{state:{count:0}}
})
vm.$mount("#app")
```

* vm.$nextTick();

```
vm.arr=[4,3,5,6]
vm.arr=[7,9,0]
vm.$nextTick(()=>{
    console.log(vm.$el.innerHTML)
    console.log(vm.$options)
})
```
